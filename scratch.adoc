  sed ':a;N;$!ba;s/\n/<text>/g'

where:

* :a       - create a label 'a'
* N        - append the next line to the pattern space
* $!       - if not the last line
* ba       - branch (go to) label 'a'
* s        - substitute
* /\n/     - regex for new line
* /<text>/ - with text "<text>"
* g        - global match (as many times as it can)
